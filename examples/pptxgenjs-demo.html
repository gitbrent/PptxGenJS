PptxGenJS - JavaScript PowerPoint Framework - Presentation Demos
DEMO: Basic Slide 
// STEP 1: Create a new Presentation
var pptx = new PptxGenJS();

// STEP 2: Add a Slide to the new Presentation
var slide = pptx.addNewSlide();

// STEP 3: Add any objects (table, text, shape or images) to the Slide
// In this case, we will use what you enter in the textbox below
slide.addText( $('#demo-basic-text').val(), {x:1.0, y:1.0, font_size:42, color:'00FF00'} );

// STEP 4: Send the PPTX Presentation to the user, using your choice of file name
pptx.save('PptxGenJs-Basic-Slide-Demo');
